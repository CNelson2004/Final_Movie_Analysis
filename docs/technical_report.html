Technical Report

Project Summary, Methodology, and Findings

Executive Summary

Project Objectives: The objective of this project is to find what features determine high grossing movies. In other words, how would I make a movie, and when would I release it in order to return the highest amount of revenue?

Key Findings: In total, we found that making a movie that appeals to everyone, is released in the summer, and has a high starting budget, results in the highest revenue. Specifically, making a PG movie using Digital Animation in the genre of Adventure or Action, which appeals to those inside and outside the US will result in the highest overall revenue. This movie should be made with the largest production budget possible, (to get the best talent), and should be released in the summer, specifically on whatever weekend has the best chance of creating the highest opening weekend revenue possible.

Actionable Recommendations: Create a movie as described in the Key Findings above.

Project Context

Motivation: Identify best way to create a movie, and the best time to release a movie to maximize revenue.

Stakeholders: Business/Board of Trustees/Boss

Success Criteria: Identify distinct features that are important for overall movie revenue

Limitations: The categorical and numerical analysis for correlation can’t show causation so we don’t know if adjusting one will directly influence the other. Additionally, none of the results shown significance, this means that the variance of revenue in any of our analyses is not different form the average mean.

Open Questions:

Future Experiments: In order to reduce the amount of variance it would be smart to separate data more, an example might be only action movies in winter compared to adventure movies in winter. This would hopefully reduce the variance and create a more accurate analysis as well as show any influence between variables, for example if R movies earning less revenue because most of them were released in winter. An analysis of movie ratings for these movies would also be informative as to what watchers value and there for what you could charge more for.
